Armazenamento de E-mail e senha;
É necessario um cadastro aonde ficará armazenado o Nome, E-mail, Senha e a rede social derivada das informações.
No index deverá aparecer os seguintes campos: E-mail, Senha(deve conter uma caixa para enviar os dados) e cadastrar usuario.
Na pagina de 'cadastrar usuario' deve reter as informações de E-mail e senha para acessar o site.
No site aparecera em formas de card as rede socias cadastradas, com seu E-mail e a senha ; E um botão para cadastrar as suas redes sociais.

# Objetivos do Projeto

### Backend

[] construir o cadastro do usuario do
[] listar os E-mail e as senhas cadastradas do usuario
[] listar as redes sociais cadastradas do usuario
[] cadastrar nova senhas cadastradas

### Frontend
